# Mein Portfolio – Vladyslav Kaplii

Willkommen auf meiner Portfolio-Seite! Hier präsentiere ich meine Projekte, meine Fähigkeiten und meine Leidenschaft für moderne Webentwicklung.

## 🚀 Über das Projekt

Diese Website dient als meine persönliche Präsentationsplattform, um mich für eine Ausbildung zum Fachinformatiker für Anwendungsentwicklung zu bewerben. Sie zeigt meine technischen Fertigkeiten, meine Design-Ideen und meine Fähigkeit, benutzerfreundliche Web-Anwendungen zu entwickeln.

## 🛠 Technologien

Das Projekt wurde mit modernen Web-Standards umgesetzt:

- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Besonderheiten:** Responsive Design, saubere Code-Struktur, clientseitige Logik.

## 📂 Enthaltene Projekte

Auf meiner Seite findest du unter anderem:

1. **WortKarten:** Ein interaktives Tool zum Sprachenlernen mit Flashcard-Logik.
2. **GehaltCheck:** Ein präziser Gehaltsrechner, der speziell auf das deutsche Steuersystem angepasst ist.
3. **KI-Spielideen-Generator:** Ein kreatives Tool für Spieleentwickler, das auf Basis von KI einzigartige Unity-Spielkonzepte erstellt.

## 📈 Ziele & Motivation

Mein Ziel ist es, analytische Probleme durch sauberen Code zu lösen. Ich bin motiviert, aktiv an realen Webprojekten mitzuarbeiten und meine Kenntnisse in einem professionellen Umfeld einzubringen.
